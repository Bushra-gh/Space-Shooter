# Space Shooter

A 2D space shooter game made with Godot.

## About

This project was created while learning Godot and GDScript. The game allows the player to move a spaceship, shoot lasers, and destroy incoming meteors.

## Controls

- Arrow Keys - Move
- Space - Shoot

## How to Run

1. Clone the repository.
2. Open the project in Godot.
3. Run the main scene.
